# Public Release Note — Kernel Paper v1

We are releasing a kernel-only public paper from the broader NFC research program.

This paper is intentionally limited to the self-contained finite theorem-bearing core: kernel axioms,
observational quotients, stabilization, boundary determinacy, interface-capacity bounds, the structural
selection theorem `K_0 = 7`, and the resulting finite consequences Phase-A and LS-2.

It does **not** claim a continuum realization, gauge-sector identification, Born-rule derivation, or
standalone Yang–Mills / Navier–Stokes / gravitational closure. Those belong to separate synthesis or
bridge-program documents.

The purpose of this release is to make the most self-contained mathematical core available in a form that
can be read independently of the broader suite.
