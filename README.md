# Nested Fibrational Cosmology

Nested Fibrational Cosmology (NFC) is a mathematical-physics research program centered on a canonical seven-book spine and derived branch books. This repository hosts the PDF and TeX source files for the current public materials.

## Repository contents

### Canonical spine
- `NFC_Book_I.*` — Primitive Relational Foundations
- `NFC_Book_II.*` — Local Structure, Boundary Law, and Stabilization
- `NFC_Book_III.*` — Persistence, Transport, and Observational Transfer
- `NFC_Book_IV.*` — The Universal Source and Completion Ladder
- `NFC_Book_V.*` — Descents, Projections, and Certified Branch Legitimacy
- `NFC_Book_VI.*` — Effective Continuum and Calculus Legitimacy
- `NFC_Book_VII.*` — Canonical Closure Schema and Limits of the Spine

### Derived branch books
- `NFC_YM_Branch.*` — Yang–Mills branch
- `NFC_NS_Branch.*` — Navier–Stokes branch
- `NFC_GR_Branch.*` — Gravity / GR branch
- `NFC_SCC_Branch.*` — Structural Counterfactual Capacity branch
