# v25 — Project Lead Pilot Portfolio

v25 is a project-control milestone.

It does not add a new branch successor. Instead, it adds a ranked intake queue for future L2 branch pilots. The point is to prevent migration momentum from outrunning governance discipline.

## Policy

A new L2 pilot should be selected only when:

1. the quality gate passes,
2. the branch has a known current posture,
3. that posture reconciles with computed status,
4. the branch has an explicit frontier profile,
5. the successor checks are named before implementation.

## Current recommendation

The current intake queue recommends SCC as the next L2 pilot.

Reasons:

- SCC is architecturally central.
- It already has local governance records in the starter.
- It feeds RH and other structural-capacity descendants.
- Its migration will test conditional closure without the Clay-grade sensitivity of RH or the inheritance complexity of SM.

## Non-action

v25 does not supersede any existing book.
v25 does not migrate SCC yet.
v25 only authorizes SCC as the next prudent pilot candidate.
