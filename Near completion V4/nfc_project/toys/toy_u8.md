## A. Primitive Tokens (No Objects, No Meanings)

### Incidence Sites

Let $\mathcal{I} = \{a, b, c, d\}$ (marks only, no structure)

### Relation Tokens (8 total)

Each relation is a token $r = (S_r, T_r)$ with finite $S_r, T_r \subset \mathcal{I}$.

**Backbone candidates (level 0):**
- $b_1 := (\{a\}, \{b\})$
- $b_2 := (\{b\}, \{c\})$
- $b_3 := (\{c\}, \{d\})$

**Halo regime tokens (level 1):**
- $h_0 := (\{a\}, \{d\})$
- $h_1 := (\{a\}, \{d\})$ (distinct token, same incidence as $h_0$)

**Defect tokens (level 1):**
- $\delta^{(1)} := (\{b\}, \{b\})$ (sub-gap candidate)
- $\delta^{(2)} := (\{c\}, \{c\})$ (super-gap candidate)

**Witness context (level 1):**
- $w := (\{d\}, \{d\})$

**No identity element is assumed.**

## B. Typed Partial Composition

### Admissibility Rule

Incidence-composition is defined **only when** $T_r \cap S_{r'} \neq \varnothing$, and then:
$$r' \circ r := (S_r, T_{r'})$$

This is **partial**; many pairs don't compose.

### Key Composites

- $b_2 \circ b_1 = (\{a\}, \{c\})$
- $b_3 \circ b_2 \circ b_1 = (\{a\}, \{d\})$

So $h_0$ and $h_1$ match the same incidence pattern as the backbone composite‚Äîbut they remain **distinct tokens** until compression.

## C. Stratification + Projection

**Levels**: $\ell(b_i) = 0$; $\ell(h_0) = \ell(h_1) = \ell(\delta^{(1)}) = \ell(\delta^{(2)}) = \ell(w) = 1$

## E. Witness Families (Fine vs. Coarse)

### Fine Witness $W_f = \{w\}$

$W_f$ reports which halo token is present when probed in admissible contexts.

**Behavior**:
- Distinguishes $h_0$ from $h_1$ at token level
- Reports distinct values for distinct tokens

### Coarse Witness $W_c = \{\}$ (or witness reporting only projected base)

$W_c$ **cannot** distinguish $h_0$ vs $h_1$; it only "sees" their shared projection.

**Behavior**:
- $\kappa_{f \to c}(h_0) = \kappa_{f \to c}(h_1) = C$ (single coarse class)

## F. Gap Threshold and Coherent Pocket

Set **witness gap threshold** $\gamma = 2$ over defect alphabet $\{\delta^{(1)}, \delta^{(2)}\}$:

- $\delta^{(1)}$ is **sub-gap**: Invisible to fine witness except as harmless micro-variation
- $\delta^{(2)}$ is **super-gap**: Witness-visible; can cause regime switching

### Coherent Pocket Statement (Toy-Level)

> As long as only $\delta^{(1)}$ is admissible, the evolution of the halo under admissible extension remains inside the $W_c$-equivalence class $C$, and $W_f$ sees no stable regime change.

This is "coherence": multiplicity exists at token level, but is not forced into public branching.

---

## G. Bounded Leakage / Drift

Allow repeated admissible extensions where $\delta^{(2)}$ sometimes appears.

### Canon-Level Control Statement

Leakage probability is bounded by gap-controlled functional and defect strength. Protected drift is quadratic in defect strength.

### Toy Interpretation

Interpret "extension depth" as parameter $T$ in leakage/drift bounds (application choice, not primitive):

- While defect budget $\int \delta(t)^2 dt$ stays small relative to $\gamma$: coherent pocket persists
- As budget grows: drift/leakage increases but remains bounded (no catastrophic backreaction‚ÄîPASS forbids halo‚Üíbackbone feedback)

---

## H. Classical Fixed Point

Under coarse witness $W_c$:

- Both $h_0$ and $h_1$ always compress to same public class $C$
- Even when super-gap defects occur, public report is stable once witness can't resolve branching

**This is the classical fixed point**: stable public regime under coarse-graining, despite micro-branching under $W_f$.
