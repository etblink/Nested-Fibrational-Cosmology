The Toggle-Complete (TC) condition: the admissible generator class includes toggle operations that cover all collar positions. Precisely: for every collar position x in del_rho(U), there exists an admissible generator E_x that toggles the state at x while leaving all other positions unchanged. TC enables the orbit argument that shows collar-type transitions are fully achievable, which is required for the LS-2 proof. TC is proved unconditionally from the toggle-complete standing rule (thm:II.TC-from-sr).
