A branch candidate is a pair (C, F) where C is a licensed target category and F: POT -> C is a declared realization functor. The target category C must be declared before the branch book begins. The realization functor F must be licensed by a bridge theorem establishing that F preserves the source structure. A branch candidate is the minimal declaration required before any endpoint claim can be made; without (C, F), there is no canonical locus for the endpoint datum.
