An admissible class is a set of external processes on a configuration space (Omega, L) satisfying four axioms: (A1) Closure under typed composition — if E and E' are admissible and type-compatible, their composition E';E is admissible; (A2) Structural invariance — if two configurations are structurally equivalent (same signature), any admissible process applies identically to both; (A3) Base-signature accountability — every admissible process declares its base signature; (A4) Finite output control — the set of outputs of any admissible process on any finite input is finite. These four axioms define the boundary of legitimate structural operations in the NFC source framework.
