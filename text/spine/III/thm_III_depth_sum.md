Depth-Sum Convergence [U]: for any Van Hove exhaustion (U_n) with |U_n| -> inf and |del(U_n)|/|U_n| -> 0, and any decay rate mu > 0: (1/|U_n|) * sum_{d>=0} |del_d(U_n)| * e^{-mu*d} -> 0 as n -> inf. Here del_d(U_n) is the set of sites at distance exactly d from the boundary. Proof: |del_d(U_n)| <= |del(U_n)| * D^d where D is the maximum degree; split the sum at d_0 = floor(log|U_n|/mu); the tail is O(|del(U_n)|/|U_n|) -> 0; the head is o(|U_n|) by the Van Hove condition. This theorem is branch-agnostic: it uses only the Van Hove exhaustion geometry. It provides the convergence input for the NS branch's weighted-boundary transport bounds (the weighted sum of boundary contributions decays to zero as the region grows).
